Для проекта нужны:

PHP_8.0
MySQL-8.0-Win10
Apache_2.4-PHP_8.0

Чтобы запустить проект нужно выполнить комманду: 
composer install

Затем настройте env файл 

После этого выполнить комманды:

php artisan key:generate
php artisan db:seed
